WP Time Slots Extended is used to provided some extended settings and features for the WP Time Slots Booking Form

== Description ==

The following featured have been extended with this plugin and usually are available through the "Extended Settings" submenu.

* Provide a custom email notification to the user when status is set to APPROVED
* Provide a custom email notification to the user when status is set to CANCELLED

== Installation ==

Download the zip from github and extract its content into the plugins directory.
Make sure the "WP Time Slots Booking Form" plugin is already installed.
